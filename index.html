Tutorial Membuat Akun BriMo

    Langkah-Langkah

  1. Download Aplikasi BriMo di App Store
  2. Buka Aplikasi BriMo
  3. Pilih Menu "Belum Punya Akun BriMo"
  4. Jika Anda Sudah Memiliki Akun Bri,pilih "Ya, Saya Punya Rekening BRI"
  5. Siapkan Dokumen Yang diperlukan
  6. Klik "Daftar BriMo"
  7. Ambil Foto KTP
  8. Cek Data Diri
  9. Verifikasi Nomor Handphone Yang Dikirim Melalui SMS Atau Whatshapp
  10. Verifikasi Email
  11. Lakukan Perekaman Wajah Sesuai Instruksi
  12. Buat User Name Dan Password BriMo
  13. Buat Pin BriMo
  14. Pendaftaran Berhasil

    Tips Aman

  • Gunakan Jaringan Internet Yang Aman,Hindari Wifi Publik
  • Buat User Id Dan Password Yang Kuat Dengan Menggunakan Kombinasi Yang Tidak Mudah Untuk Ditebak
  • Rahasiakan User Id,Password,Dan PIN
  • Ubah Password Dan PIN Secara Teratur
  • Periksa Mutasi Rekening Secara Rutin
  • Aktifkan SMS Notifikasi Di Kantor Bri Terdekat

Tutorial Membuat Akun Dana

    Langkah-Langkah

  1. Download dan install Aplikasi Dana
  2. Buka Aplikasi Dan
  3. Registrasi Akun Baru
  4. Verifikasi Nomor Baru
  5. Isi Data Pribadi
  6. Buat Pin Dana
  7. Aktifasi Akun
  8. Verifikasi KTP
  9. Mulai Menggunakan Dana

    Tips Aman

  • Aktifkan Fitur Keamanan Tambahan
  • Jaga Kerahasiaan PIN
  • Priksa Transaksi Secara Berkala
